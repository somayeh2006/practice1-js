Project README
Order Processing System
A simple JavaScript practice project that simulates processing online store orders.



Purpose
Practice arrays of objects
Use the while loop
Apply conditional logic
Manage stock availability
Work with functions



Logic Overview
Each order includes:
price, paid, delivered, customertype
An order is shipped if:
It is paid
It has not been delivered
Stock is available
VIP customers always get shipping.
Normal customers get shipping only if the order price meets the free shipping limit.


Output
Index of shipped orders
Total shipped and non-shipped orders
Remaining stock
Result of order validation function

Concepts Used
Arrays & Objects
while loop
if / else
break
